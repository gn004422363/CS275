## Introduction

This program display a HTML web page invoke the weather map web service by using jQuery and Ajax. 

## Files

* ml3546_hw2.html

## Installation

Open the .html file with any web browser.


## Using the Webpage

User can input an API key in the first text field and zip code in the second text field. Then click submit. The API will respond and send back required data. The result will display in the bottom of the page with a table that shows 5 days / 3 hours forecasts.
